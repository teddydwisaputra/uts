Jelaskan kapan saat yang tepat menggunakan metode byClassName jelaskan kelebihan dan kekurangannya.

Code javascript :

//by className const p1 = document.getElementsByClassName('p1'); p1[0].style.backgroundColor='yellow';

Penjelasan sintaks sebagai berikut :

document adalah object dari dokumen html tersebut.

document mempunyai method getElementBy TagClassName('p1') yang di gunakan untuk mengakses elemen berdasarkan TagName.

Sintaks document.getElementBy TagClassName('p1') sudah mengacu kepada html collection


Kelebihan:

Kita tidak harus membuat “tagclassname” berbeda pada setiap dokumen

Kekurangan:

Saat kita ingin melakukan manipulasi dom kita harus melakukan spesifikasi lagi contoh p1[0].style.backgroundColor='yellow'; p1 sebagai tagnmae kurang spesifik karena berbentuk collection diberi angka urutan [0] [1] dan seterusnya.

Note : (bisa tanpa spesifikasi tapi menggunakan looping)
